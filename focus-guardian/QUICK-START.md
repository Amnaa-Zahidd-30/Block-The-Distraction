## ðŸ’¬ Common Questions

**Q: Do I need to keep the app open all day?**
A: Yes, in a browser tab. Pin it so it stays open.

**Q: What if I complete a task early?**
A: Check it off! All apps unlock when all tasks are done.

**Q: Can I add tasks mid-day?**
A: Currently no - plan everything in the morning.

**Q: The warning screen doesn't show up?**
A: Make sure you're:
  1. In an active task time block right now
  2. Using the web browser (not mobile app)
  3. On a blocked site

**Q: Can I use this on my phone?**
A: Yes! Open in mobile browser. But it works better on computer.

**Q: Will my tasks save overnight?**
A: No - currently resets daily. Fresh start each morning!
